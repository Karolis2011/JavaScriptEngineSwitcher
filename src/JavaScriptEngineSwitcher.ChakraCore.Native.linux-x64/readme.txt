   ===========
   DESCRIPTION
   ===========
   This package complements the JavaScriptEngineSwitcher.ChakraCore package and
   contains the native implementation of ChakraCore version 1.11.22 with the Tomáš
   Deml's patch for Linux (x64).

   This package is only compatible with .NET Core.

   =============
   RELEASE NOTES
   =============
   ChakraCore was updated to version 1.11.22 (Tomáš Deml's patch applied).
